============
Installation
============
For **Linux or Mac OS** users:
==============================

.. code-block:: sh

   pip3 install libsodium

For **Windows** users:
======================

.. code-block:: sh

   pip install libsodium

Or manualy install via git:

.. code-block:: sh

   cd <site-packages>
   git clone https://github.com/zykron1/sodium
   mv sodium libsodium
